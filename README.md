Personalized web page for a course.

1) The best way to show the web page is to set up a local server, with for example Visual Studio. Download the Visual Studio plugin by Ritwick Dey:

https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer

Press Go Live in the bottom right of Visual Studio.

2) A simpler way that may not always work.
   
To run the web page put all files in the same folder and double click index.html.
